## Installation 


To install my Neovim setup, simply run this script:

```bash
git clone https://github.com/NvChad/starter ~/.config/nvim && nvim

git clone https://github.com/chocoOnEstrogen/dotfiles.git 

cp -r dotfiles/machines/personal/applications/neovim ~/.config/nvim
```
